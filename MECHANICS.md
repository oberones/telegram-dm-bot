# Arbitration Rules – Mimir Writers’ Room (D&D 5e Arena)

This file defines how we resolve deadlocked creative decisions using Dungeons & Dragons 5e–style arena combat between writer-avatars.

## Core Rules

1. **Who Has Characters**
   - Each human writer gets a D&D 5e character:
     - **Ignus VanMule (Steven)**
     - **Dilbo Shaggins**
     - **pulsar**
     - **Robert Strength**
   - Daneel (the AI) is the referee and does **not** get a combatant.

2. **When Arbitration Triggers**
   - If a decision comes down to **equal votes on each side** after normal discussion, we trigger an **Arbitration Match**.
   - Only the characters belonging to the humans on each side of the disagreement enter the arena.
   - If it’s truly all-vs-all, the match can be **free-for-all**; otherwise, it’s **team vs. team**.

3. **Arbitration Match Format**
   - System: **Dungeons & Dragons 5e** (simplified / fast resolution when needed).
   - Mode: **Arena fight to the death** (physical or virtual, narrative-only; no real people harmed, obviously).
   - Initiative, attack rolls, damage, saving throws, etc., follow 5e rules.
   - Environment is assumed to be a **neutral arena** unless we explicitly theme it.

4. **Death, Respawn, and Naming**
   - When a character dies in arbitration:
     - They are considered **dead for that match**.
     - For future arbitrations, they **respawn** as a new iteration with a suffix:
       - Example: *Rook* → *Rook II* → *Rook III*, etc.
   - The **name + Roman numeral** tracks that character’s death history.

5. **Experience and Leveling**
   - Any time a character **wins or loses** an arbitration fight, they **gain 1 level**.
   - Leveling follows **standard D&D 5e rules** for the chosen class:
     - Increase Hit Points, class features, spell slots, etc., as appropriate.
   - Daneel (the AI) is responsible for applying level-ups correctly.

6. **Character Changes**
   - A player can request **changes to their character** at any time **between** arbitrations, as long as the result is **rules-legal 5e**.
   - No custom mega-cheese items like an “Uber Slaying Longsword of Dick and Balls.”
     - Homebrew is allowed only if the group and Daneel agree it’s roughly balanced.

7. **Equipment & Spells**
   - Baseline characters start with **standard class equipment** as per D&D 5e.
   - After players pick a class, Daneel will:
     - Roll up a **standard build** quickly.
     - Choose **standard gear** and a spell loadout (if applicable) **biased toward damage and one-on-one combat**.
   - Players can later tune their loadouts within 5e rules.

8. **Names**
   - Each player must provide a **character name**.
   - That name (plus the Roman numeral suffix on respawns) will be used in arbitration logs.

The group opted to **pause the full D&D arbitration system** to save tokens and complexity. For now, deadlocked decisions will fall back to simpler methods (e.g., a single d20 roll and a pre-agreed mapping) instead of arena fights.

Existing content (rules and Captain Argot LoVallo’s stat block) remains here for possible future use, but character creation tasks for Ignus / Dilbo / pulsar are on hold.

Once/if you all decide to revive the arena system, we can re-open these tasks and finish building everyone’s characters.
